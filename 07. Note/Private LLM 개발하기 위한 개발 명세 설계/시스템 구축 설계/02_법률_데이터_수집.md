## 2. 공개 데이터 소스

### 2.1 법률 조문

#### 국가법령정보센터 (law.go.kr)

| 항목 | 내용 |
|------|------|
| URL | https://www.law.go.kr |
| 제공 데이터 | 대한민국 모든 현행 법률 원문 |
| 수집 대상 | 민법, 민사소송법, 민사집행법, 가사소송법 등 |
| 포맷 | HTML, XML (Open API 제공) |
| 비용 | 무료 |

**Open API 활용:**

```
# API 키 발급: law.go.kr > 열린 마당 > Open API
# 법령 검색 API 엔드포인트
https://www.law.go.kr/DRF/lawSearch.do?OC={API_KEY}&target=law&type=XML&query=민법
```

**수집 대상 법률 목록:**

```
1. 민법 (전 5편, 1,118조)
2. 민사소송법
3. 민사집행법
4. 가사소송법
5. 주택임대차보호법
6. 상가건물 임대차보호법
7. 부동산 실권리자명의 등기에 관한 법률
8. 이자제한법
9. 소비자기본법
10. 전자상거래 등에서의 소비자보호에 관한 법률
```

#### 수집 스크립트 예시

```python
import requests
import xml.etree.ElementTree as ET
from pathlib import Path

API_KEY = "YOUR_API_KEY"
BASE_URL = "https://www.law.go.kr/DRF/lawService.do"

def fetch_law(mst_id: str, output_dir: Path) -> None:
    """법령 전문 다운로드."""
    params = {
        "OC": API_KEY,
        "target": "law",
        "MST": mst_id,
        "type": "XML",
    }
    resp = requests.get(BASE_URL, params=params, timeout=30)
    resp.raise_for_status()

    root = ET.fromstring(resp.content)
    law_name = root.findtext(".//법령명_한글", default="unknown")

    output_path = output_dir / f"{law_name}.xml"
    output_path.write_bytes(resp.content)
    print(f"저장 완료: {output_path}")
```

#### 직접 생성 (Synthetic Data)

고품질 데이터 보충을 위해 기존 판례/조문 기반으로 Q&A 쌍을 생성할 수 있습니다:

```
원본 판례 요약 → "이 상황에서 어떻게 해야 하나요?" 형태의 질문 생성
법조문 → "~한 경우 법적으로 어떤 보호를 받을 수 있나요?" 형태의 질문 생성
```

## 3. 데이터 수집 시 주의사항

### 3.1 저작권 및 라이선스

| 데이터 | 저작권 상태 | 사용 가능 여부 |
|--------|-----------|--------------|
| 법률 조문 | 저작권 없음 (공공저작물) | 자유 사용 |
| 대법원 판례 | 저작권 없음 (공공저작물) | 자유 사용 |
| AI Hub 데이터셋 | 공공누리 라이선스 | 조건부 사용 (라이선스 확인 필수) |
| 법률구조공단 상담 | 공공저작물 | 출처 표시 후 사용 |
| 민간 법률 상담 | 저작권 있음 | 이용약관 확인 필수 |
| 법학 논문/교과서 | 저작권 있음 | 허락 없이 사용 불가 |

### 3.2 개인정보 처리

```
반드시 제거해야 할 개인정보:
- 실명 → "갑", "을", "원고", "피고"로 대체
- 주민등록번호 → 완전 삭제
- 주소 → "서울시 ○○구" 수준으로 일반화
- 전화번호 → 완전 삭제
- 계좌번호 → 완전 삭제
- 사건번호는 유지 가능 (공개 판례의 경우)
```

### 3.3 데이터 품질 기준

```
✅ 포함 기준:
- 민사 분야에 해당하는 내용
- 질문과 답변이 완결된 형태
- 한국어로 작성된 내용
- 현행법 기준 유효한 내용

❌ 제외 기준:
- 형사/행정/헌법 사건 (도메인 외)
- 답변이 불완전하거나 모호한 사례
- 폐지/개정된 법률 기반 답변 (별도 표시 없는 경우)
- 개인정보가 완전히 제거되지 않은 데이터
```
