## 2. RAG용 문서 전처리

### 2.1 법률 조문 정제

**원본 형태 (XML/HTML에서 추출 후):**
```
민법
제1편 총칙
  제1장 통칙
    제1조(법원) 민사에 관하여 법률에 규정이 없으면 관습법에 의하고 관습법이 없으면 조리에 의한다.
    제2조(신의성실) ①권리의 행사와 의무의 이행은 신의에 좇아 성실히 하여야 한다.
    ②권리는 남용하지 못한다.
```

**정제 후 (조항 단위 분리):**
```
제1조(법원) 민사에 관하여 법률에 규정이 없으면 관습법에 의하고 관습법이 없으면 조리에 의한다.
[출처: 민법 제1편 총칙 > 제1장 통칙]
```

**정제 스크립트 가이드:**

```python
import re
from pathlib import Path

def clean_statute(raw_text: str, law_name: str) -> list[dict[str, str]]:
    """법률 조문을 조항 단위로 분리하여 정제."""
    articles = []
    # 조항 패턴: 제N조, 제N조의N
    pattern = re.compile(r'(제\d+조(?:의\d+)?(?:\([^)]+\))?)')
    parts = pattern.split(raw_text)

    current_article = ""
    for part in parts:
        if pattern.match(part):
            if current_article.strip():
                articles.append(current_article.strip())
            current_article = part
        else:
            current_article += part

    if current_article.strip():
        articles.append(current_article.strip())

    return [
        {
            "title": f"{law_name} {art.split(')')[0]})" if ')' in art else law_name,
            "doc_type": "법조문",
            "content": art,
        }
        for art in articles
    ]
```

### 2.2 판례 정제

**판례 구조 분석:**

```
판례 원문 구조:
├── 사건번호: 2023다12345
├── 선고일자: 2023.05.15
├── 법원: 대법원
├── 사건유형: 손해배상(기)
├── 판시사항 (핵심 요약)
├── 판결요지 (법적 판단 요약)
├── 참조조문 (관련 법조문)
├── 참조판례
├── 전문 (사실관계 + 법적 판단 전체)
└── 주문 (결론)
```

**정제 원칙:**
```
1. 판시사항 + 판결요지 → 별도 청크 (핵심 검색 대상)
2. 전문 → 512 토큰 단위 청킹 (오버랩 64)
3. 메타데이터 보존: 사건번호, 법원, 선고일, 사건유형
4. 참조조문 → 메타데이터에 포함
```

```python
def clean_case(raw_text: str) -> dict[str, str]:
    """판례 원문을 정제된 형태로 변환."""
    # 사건번호 추출
    case_num_match = re.search(r'(\d{4}[가-힣]+\d+)', raw_text)
    case_num = case_num_match.group(1) if case_num_match else "미상"

    # 불필요한 요소 제거
    cleaned = raw_text
    cleaned = re.sub(r'\s+', ' ', cleaned)       # 연속 공백 정리
    cleaned = re.sub(r'【[^】]*】', '', cleaned)   # 【】 태그 제거

    return {
        "title": f"판례 {case_num}",
        "doc_type": "판례",
        "content": cleaned.strip(),
    }
```

def generate_qa_from_case(case: dict[str, str]) -> dict[str, str]:
