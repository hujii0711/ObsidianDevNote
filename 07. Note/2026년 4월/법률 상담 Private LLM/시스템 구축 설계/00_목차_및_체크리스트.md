## 마스터 체크리스트

### Phase 1: 환경 준비

- [ ] MacBook Pro M4 48GB 확인
- [ ] macOS 14 이상 업데이트
- [ ] Xcode Command Line Tools 설치
- [ ] Python 3.11+ 설치
- [ ] 가상환경 생성 및 의존성 설치
- [ ] MLX 동작 확인 (`mx.default_device()` = gpu)
- [ ] EEVE-Korean-10.8B-4bit 다운로드 (~8GB)
- [ ] BGE-M3 임베딩 모델 다운로드 (~2.2GB)
- [ ] ko-sroberta 리랭킹 모델 다운로드 (~1.1GB)
- [ ] 모델 로딩 및 한국어 응답 테스트 통과
- [ ] 디스크 여유 공간 100GB 이상 확인

### Phase 2: 데이터 준비

- [ ] 국가법령정보센터 Open API 키 발급
- [ ] 민법 전문 다운로드
- [ ] 민사소송법 전문 다운로드
- [ ] 주요 특별법 10개 이상 다운로드
- [ ] AI Hub 법률 데이터셋 신청 및 다운로드
- [ ] 대법원 민사 판례 5,000건 이상 수집
- [ ] 법률 상담 Q&A 10,000건 이상 수집
- [ ] 법률 용어 사전 500개 이상 수집
- [ ] 개인정보 비식별화 처리 완료
- [ ] RAG용 문서 정제 (조항 단위 분리, 판례 정제)
- [ ] Fine-tuning용 데이터 JSONL 변환
- [ ] train/valid/test 분할 완료

### Phase 3: 시스템 구축

- [ ] RAG 인덱싱 실행 성공
- [ ] 벡터DB 문서 수 확인 (1,000건 이상)
- [ ] 검색 테스트 통과 (관련 문서 상위 노출)
- [ ] 검색 응답 시간 < 1초
- [ ] QLoRA 학습 실행 성공 (loss 감소 확인)
- [ ] 학습 중 메모리 안정 (OOM 없음)
- [ ] 모델 병합 완료
- [ ] Before/After 품질 비교 완료
- [ ] 테스트 스위트 통과 (`pytest tests/ -v`)

### Phase 4: 배포

- [ ] API 서버 기동 및 헬스체크 정상
- [ ] Streamlit UI 접속 및 채팅 테스트 성공
- [ ] 스트리밍 응답 정상 동작
- [ ] 면책 조항 모든 응답에 포함 확인
- [ ] Docker 빌드 및 실행 성공 (선택)
- [ ] 백업 스크립트 설정
- [ ] 로그 모니터링 설정

## 예상 전체 소요 기간

| Phase | 기간 | 병목 요소 |
|-------|------|----------|
| 환경 준비 | 1일 | 모델 다운로드 네트워크 속도 |

| 배포 | 1일 | - |
| **합계** | **약 2~3주** | |

> 데이터 수집이 전체 일정의 절반 이상을 차지합니다. 데이터 수집과 환경 구축을 병렬로 진행하면 전체 기간을 단축할 수 있습니다.
