## 학습 완료 자가 점검

### Level 1: 개념 이해 (Week 1 목표)

```
□ "Transformer가 뭐예요?"를 비개발자에게 설명할 수 있다
□ "LoRA가 Full Fine-tuning 대비 무엇이 좋은지" 설명할 수 있다
□ "RAG가 왜 환각 방지에 도움이 되는지" 설명할 수 있다
□ "민법 제750조"가 무슨 내용인지 알고 있다
□ "소멸시효"가 무엇인지 일반인에게 설명할 수 있다
```

### Level 2: 기술 실습 (Week 2 목표)

```
□ BGE-M3으로 한국어 문장 임베딩을 생성하고 유사도를 계산할 수 있다
□ ChromaDB에 문서를 저장하고 검색할 수 있다
□ mlx-lm으로 모델을 로딩하고 텍스트를 생성할 수 있다
□ FastAPI로 POST 엔드포인트를 만들 수 있다
□ async/await으로 비동기 함수를 작성할 수 있다
```

### Level 3: 통합 구현 (Week 3 목표)

```
□ RAG 검색 결과를 프롬프트에 삽입하여 LLM 응답을 생성할 수 있다
□ SSE로 스트리밍 응답을 구현할 수 있다
□ 학습 Loss 곡선을 보고 과적합 여부를 판단할 수 있다
□ 모델 응답의 법적 정확성을 기초 수준에서 검증할 수 있다
□ 전체 시스템을 로컬에서 기동하고 채팅 테스트를 할 수 있다
```
