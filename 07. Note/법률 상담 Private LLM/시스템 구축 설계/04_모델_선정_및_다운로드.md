## 2. 임베딩 모델

### 2.1 선정: BGE-M3

| 항목 | 내용 |
|------|------|
| 모델명 | BAAI/bge-m3 |
| 차원 | 1024 |
| 다국어 지원 | 100+ 언어 (한국어 포함) |
| 크기 | ~2.2GB |
| 특징 | Dense + Sparse + ColBERT 멀티벡터 |

**선정 이유:**
- 한국어 임베딩 성능 최상위
- 법률 문서의 긴 컨텍스트 처리 가능 (max 8192 토큰)
- LlamaIndex 네이티브 통합 지원

### 2.2 다운로드 및 검증

```python
from sentence_transformers import SentenceTransformer

# 자동 다운로드 (최초 실행 시 ~2.2GB)
model = SentenceTransformer("BAAI/bge-m3")

# 한국어 임베딩 테스트
sentences = [
    "��대차 보증금 반환 청구",
    "전세 보증금을 돌���받고 싶습니다",
    "형사 고소장 작성 방법",
]
embeddings = model.encode(sentences)

# 유사도 확인 (임대차 관련 2개가 높아야 함)
from sklearn.metrics.pairwise import cosine_similarity
sim = cosine_similarity(embeddings)
print(f"임대차-전세 유사도: {sim[0][1]:.4f}")  # 높아야 함
print(f"임대차-형사 유사도: {sim[0][2]:.4f}")  # 낮아야 함
```

## 3. 리랭킹 모델

### 3.1 선정: ko-sroberta-multitask

| 항목 | 내용 |
|------|------|
| 모델명 | jhgan/ko-sroberta-multitask |
| 용도 | 한국어 문장 유사도 기반 리랭킹 |
| 크기 | ~1.1GB |
| 사용 방식 | CrossEncoder |

### 3.2 다운로드 및 검증

```python
from sentence_transformers import CrossEncoder

# 자동 다운로드
reranker = CrossEncoder("jhgan/ko-sroberta-multitask")

# 리랭킹 테스트
query = "전세 보증금 반환 방법"
documents = [
    "임대차보호법에 따른 보증금 반환 절차를 설명합니다.",     # 관련 높음
    "형사 고소장 작성 시 주���사항을 안내��니다.",              # 관련 낮음
    "임차인의 보증금 반환 청구권은 소멸시효가 10년입니다.",    # 관련 높음
]

pairs = [(query, doc) for doc in documents]
scores = reranker.predict(pairs)

for doc, score in sorted(zip(documents, scores), key=lambda x: -x[1]):
    print(f"[{score:.4f}] {doc[:40]}...")
```
