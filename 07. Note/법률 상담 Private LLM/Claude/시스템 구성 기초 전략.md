## 3. 기술 스택

### 3.1 파운데이션 모델

| 후보 모델 | 파라미터 | 양자화 시 VRAM | 적합성 |
|-----------|---------|---------------|--------|
| **Llama 3.1 8B** | 8B | ~6GB (Q4) | 한국어 성능 우수, 커뮤니티 활발 |
| Mistral 7B | 7B | ~5GB (Q4) | 추론 속도 빠름, 한국어 보완 필요 |
| EEVE-Korean-10.8B | 10.8B | ~8GB (Q4) | 한국어 특화, M4 48GB 충분 |

**권장**: EEVE-Korean-10.8B (한국어 법률 도메인 특성상 한국어 성능 우선)

### 3.2 추론 엔진

| 도구 | 역할 |
|------|------|
| **MLX** | Apple Silicon 네이티브 추론 (M4 최적화) |
| llama.cpp | 대안 추론 엔진 (GGUF 포맷) |

### 3.3 Fine-tuning

| 도구 | 역할 |
|------|------|
| **MLX-LM** | Apple Silicon에서 LoRA/QLoRA 파인튜닝 |
| Unsloth | 메모리 효율적 파인튜닝 (대안) |

### 3.4 RAG Pipeline

| 도구 | 역할 |
|------|------|
| **LlamaIndex** | 문서 파싱, 인덱싱, 검색 오케스트레이션 |
| ChromaDB | 벡터 저장소 (로컬, 경량) |
| **BGE-M3** | 임베딩 모델 (다국어, 한국어 우수) |
| KoSentenceBERT | 한국어 리랭킹 |

### 3.5 서빙 / 인프라

## 4. RAG + Fine-tuning 혼용 전략

### 4.1 역할 분담

```
┌──────────────────────────┬──────────────────────────┐
│      Fine-tuning         │         RAG              │
├──────────────────────────┼──────────────────────────┤
│ 법률 용어 이해           │ 최신 판례 검색           │
│ 답변 형식/톤 학습        │ 조문 원문 참조           │
│ 민사소송 절차 지식       │ 유사 사례 매칭           │
│ 한국어 법률 문맥 적응    │ 근거 출처 제시           │
└──────────────────────────┴──────────────────────────┘
```

### 4.2 처리 흐름

1. **사용자 질의 입력**
2. **Query Classifier**: 질의 유형 분류 (법률 용어 / 판례 검색 / 절차 안내 / 복합)
3. **RAG 검색**: 관련 법률 조문, 판례를 벡터 DB에서 검색
4. **컨텍스트 결합**: 검색 결과 + 시스템 프롬프트를 Fine-tuned 모델에 전달
5. **응답 생성**: Fine-tuned 모델이 법률 도메인에 맞는 톤과 형식으로 답변
6. **후처리**: 출처 표기, 면책 조항 자동 첨부

## 6. Fine-tuning 환경 (M4 48GB)

### 6.1 학습 구성

```yaml
모델: EEVE-Korean-10.8B (Q4 양자화)
방법: QLoRA
  - rank: 64
  - alpha: 128
  - target_modules: [q_proj, v_proj, k_proj, o_proj]
배치 크기: 4
Gradient Accumulation: 8
학습률: 2e-4
에폭: 3
예상 학습 시간: 8~12시간 (10K 샘플 기준)
```

### 6.2 MLX Fine-tuning 실행

```bash
# MLX-LM 설치
pip install mlx-lm

# LoRA 파인튜닝
mlx_lm.lora \
  --model mlx-community/EEVE-Korean-10.8B-4bit \
  --train \
  --data ./training_data \
  --batch-size 4 \
  --lora-layers 16 \
  --iters 1000

# 모델 병합
mlx_lm.fuse \
  --model mlx-community/EEVE-Korean-10.8B-4bit \
  --adapter-path ./adapters
```

## 10. 면책 및 제약 사항

- 본 시스템은 **법률 정보 제공** 목적이며, 공인된 법률 자문을 대체하지 않음
- 모든 응답에 면책 조항 자동 첨부
- 개인정보 포함 질의는 로컬에서만 처리 (외부 전송 없음)
- 모델 환각(hallucination) 방지를 위해 RAG 출처 기반 응답 강제
