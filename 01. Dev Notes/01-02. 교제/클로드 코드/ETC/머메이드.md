머메이드(Mermaid.js)는 텍스트 기반으로 다이어그램을 그릴 수 있는 아주 효율적인 도구입니다. 개발 문서나 기술 블로그에서 가장 자주 쓰이는 대표적인 예시들을 정리해 드릴게요.

---

## 1. 순서도 (Flowchart)

가장 기본적이면서 프로세스 로직을 표현할 때 유용합니다.

코드 스니펫

```
graph TD
    A[시작] --> B{조건 확인}
    B -- Yes --> C[승인 프로세스]
    B -- No --> D[반려]
    C --> E[종료]
    D --> E
```

- **TD/LR:** `TD`(Top Down)는 위에서 아래로, `LR`(Left Right)은 왼쪽에서 오른쪽으로 방향을 정합니다.
    

---

## 2. 시퀀스 다이어그램 (Sequence Diagram)

객체나 시스템 간의 상호작용 순서를 설명할 때 필수적입니다.

코드 스니펫
```
sequenceDiagram
    participant U as 사용자
    participant S as 서버
    participant DB as 데이터베이스

    U->>S: 로그인 요청 (ID, PW)
    S->>DB: 회원 정보 조회
    DB-->>S: 정보 일치 확인
    alt 인증 성공
        S-->>U: 로그인 성공 (Token)
    else 인증 실패
        S-->>U: 오류 메시지 전송
    end
```

---

## 3. 클래스 다이어그램 (Class Diagram)

객체 지향 프로그래밍 구조를 설계하거나 시각화할 때 사용합니다.

코드 스니펫

```
classDiagram
    class Vehicle {
        +String brand
        +run()
    }
    class Car {
        +int wheelCount
        +openTrunk()
    }
    Vehicle <|-- Car : 상속
```

---

## 4. 간트 차트 (Gantt Chart)

프로젝트 일정 관리와 타임라인을 한눈에 볼 때 유용합니다.

코드 스니펫

```
gantt
    title 프로젝트 개발 일정
    dateFormat  YYYY-MM-DD
    section 설계 단계
    요구사항 분석   :done,    des1, 2026-04-01, 2026-04-05
    UI 디자인      :active,  des2, 2026-04-06, 5d
    section 개발 단계
    백엔드 구현     :         dev1, after des2, 10d
    프론트엔드 구현  :         dev2, after des2, 10d
```

---

## 5. 엔티티 관계도 (ERD)

데이터베이스의 구조와 관계를 정의할 때 사용합니다.

코드 스니펫

```
erDiagram
    USER ||--o{ ORDER : places
    ORDER ||--|{ LINE-ITEM : contains
    USER {
        string username
        string email
    }
```
