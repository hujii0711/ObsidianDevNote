
MCP 서버가 포함 확인
```
claude mcp list
```

클로드를 실행할 때 명령줄 인자로 설정을 전달
```
claude config set <키> <값>
```

---
## 슬래시 명령어

### 세션관리
- /clear
- /compact
- /rewind: 현재 세션의 채팅 기록을 확인(esc 두번 누르면 같은 효과)
- /context: 컨텍스트 윈도에서 어떤 정보가 얼마나 차지했는지 내용을 시각화
- /export: 클로드의 출력, 현재 세션의 대화 내용을 클립보드나 파일로 내보냄
- /resume(continue): 과거의 세션 목록에서 선택하여 재개, 이전 대화에서 계속하기
  /clear를 실행해 컨텍스트를 삭제해버린 경우에도 삭제하기 전으로 돌아가고 싶을 때 사용
- /exit: 클로드 대화 모드 종료, 세션 종료

### 계정 및 인증
- /login
- /logout
- /upgrade

### 프로젝트 설정
- /add-dir: 다른 디렉터리에 있는 추가 작업 디렉터리로 지정
- /init: 기존 프로젝트에 CLAUDE.md 파일을 생성
- /memory
- /config
- /permissions: 특정 명령어 실행 허용
- /ide: IDE와의 연동 활성화
- /terminal-setup

### 모델, AI 기능
- /model: 모델 변경
- /agents: 서브 에이전트 생성 여부 확인
- /vim

### 개발 지원
- /review: 리뷰를 지시
- /pr-comments
- /bashes: 백그라운드로 보낸 Bash의 내용을 확인
- /hooks: 대화 모드에서 훅 생성
- /install-github-app
- /security-review: 코드를 커밋하기 전에 터미널에서 보안 분석을 실행(코드 베이스내 잠재적인 취약점을 검색하고 발견된 문제를 상세하게 설명)
- /mcp
- /sandbox: 샌드박스 모드로 클로드 실행
- /update: 구독 한도로 인한 플랜 전환

### 정보 표시
- /todos
- /help
- /status
- /cost: 현재 세션의 사용 현황을 확인
- /usage: 구독 플랜의 사용량 확인
- /release-notes

### UI 출력 생성
- /output-style
- /output-style:new
- /statusline

### 플러그인 및 개인 정보 보호
- /plugin
- /privacy-settings: 개인정보 보호 설정 및 화면이 표시

### 유지보수 및 문제 해결
- /doctor
- /feedback
