- Sonnet은 모델명에 1m을 붙여 100만 토큰의 컨텍스트 윈도를 유지한다.
- ripgrep은 Rust로 제작된 고속 텍스트 검색 도구이다. 클로드 코드의 파일 탐색은 ripgrep의 정규 표현식 검색에 의존한다.
- gh 명령어를 사용하면 GitHub의 원격 레포지터리를 만들거나 관리할 수 있고, 이슈 생성, 풀 리퀘스트 확인, 리뷰 등도 브라우저를 열지 않고 터미널에서 바로 처리할 수 있다.
- @: 특정 파일을 지정해 컨텍스트에 포함하여 클로드에 질문하거나 대화할 수 있다.
- 사용자 루트: ~/.claude 디렉터리는 모든 클로드 프로젝트에 영향을 미친다. 클로드 코드를 처음 설치했을 때 이 사용자 루트에 .claude 디렉터리가 저장되었으며, 설정이나 기록 등이 저장되어 있다.
- 설정을 사용자 루트에 저장하면 모든 프로젝트에 적용되는 공통 설정이 되고, 프로젝트 루트에 저장하면 해당 프로젝트에만 적용된다. 
- 대화 모드에서 Tab 키를 누르는 것만으로 사고 확장 모드 기능이 추가된다.
