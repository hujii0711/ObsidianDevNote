온보딩 마법사 시작
```
openclaw onboard
```

모델 변경
```
openclaw models set google/gemini-3-flash-preview
```

모델 별칭 등록
```
openclaw models aliases add gemini google/gemini-3-flash-preview
```

게이트웨이를 재시작(설정 적용)
```
openclaw gateway restart
```

오픈클로 컨트롤 UI 접속(http://localhost:18789)
```
openclaw dashboard
```

메인 모델 실패시 자동으로 다른 모델 전환(failover chain)
```
openclaw models fallbacks add google/gemini-3-flash-preview
```

모델 상태 확인
```
openclaw models status
```

에이전크가 개인 크롬과 완전히 분리된 전용 브라우저 환경을 사용
```
openclaw config set browser.enabled true --json
openclaw config set browser.defaultProfile "openclaw"
```

에이전트 상태 점검
```
openclaw status -all
```

에이전트 문제 진단
 ```
 openclaw doctor --fix
 ```

게이트웨이 재시작
```
openclaw gateway restart
```

설치된 스킬 목록 확인
```
openclaw skills list
```

스킬 설치
```
npx clawhub@latest install <스킬 이름>
```
설치가 끝나면 워크스페이스의 skills/ 폴더에 스킬 파일이 생긴다. 재시작 필요가 없다.

스킬이 제대로 설치했는지 확인
```
openclaw skills info <스킬 이름>
```
