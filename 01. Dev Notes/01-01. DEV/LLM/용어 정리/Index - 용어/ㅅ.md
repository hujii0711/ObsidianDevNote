- 소프트맥스(Softmax)
	소프트맥스(Softmax) 함수의 역할은 주로 다중 클래스 분류 문제에서 모델의 출력을 확률 분포로 변환하는 데 있습니다. 이를 통해 모델의 예측을 해석 가능하게 하고, 학습 및 평가에 유용한 정보를 제공합니다.
	
	1. 확률 분포 생성
	소프트맥스 함수는 모델의 출력 벡터를 각 클래스에 속할 확률로 변환합니다.
	이는 입력 벡터의 각 요소를 지수 함수로 변환한 후, 이 값들을 전체 합으로 나누어 확률을 계산합니다.
	결과적으로 각 클래스에 대한 확률이 0과 1 사이의 값이 되며, 모든 클래스 확률의 합은 1이 됩니다.
	
	2. 다중 클래스 분류 지원
	소프트맥스는 다중 클래스 분류 문제에서 특히 유용합니다.
	예를 들어, 이미지가 여러 클래스 중 하나로 분류되어야 하는 문제에서, 소프트맥스는 각 클래스에 속할 확률을 계산하여 가장 높은 확률을 가진 클래스를 예측 결과로 선택할 수 있게 합니다.
	
	3. 모델의 예측 해석 용이성 제공
	소프트맥스 함수의 출력은 확률 값으로 제공되므로, 모델의 예측을 더 직관적으로 해석할 수 있습니다.
	예를 들어, 특정 클래스에 대해 0.8의 확률을 출력하면, 해당 클래스에 속할 가능성이 높다고 판단할 수 있습니다.
	
	4. 크로스 엔트로피 손실 함수와의 결합
	소프트맥스 출력은 크로스 엔트로피 손실 함수와 자주 결합되어 사용됩니다.
	크로스 엔트로피 손실은 예측된 확률 분포와 실제 레이블 간의 차이를 측정하여, 모델의 성능을 평가하고 최적화하는 데 도움을 줍니다.
	
	5. 경사하강법과의 호환성
	소프트맥스 함수는 미분 가능하므로, 역전파(backpropagation)를 통한 경사하강법(gradient descent) 최적화에 적합합니다.
	이를 통해 모델의 파라미터를 학습하고 조정할 수 있습니다.
	
	이러한 역할을 통해 소프트맥스 함수는 딥러닝 모델이 다중 클래스 분류 문제를 효과적으로 해결할 수 있도록 돕고,  모델의 출력이 확률로 해석될 수 있게 함으로써, 예측의 신뢰도를 평가하고, 모델의 성능을 더욱 향상시킬 수 있습니다.

- 시퀀스
	딥러닝 모델인 트랜스포머(Transformer)에서 "시퀀스"는 주로 연속된 데이터 포인트의 집합을 의미하며, 이는 자연어 처리(NLP)에서 특히 중요합니다.
	트랜스포머 모델은 이러한 시퀀스를 입력으로 받아 처리하고, 또 다른 시퀀스를 출력으로 생성하는 데 특화되어 있습니다.
	시퀀스의 개념은 여러 문맥에서 다양한 의미를 가질 수 있습니다:
	
	1. 입력 시퀀스: NLP에서 입력 시퀀스는 일반적으로 단어, 토큰, 또는 문장으로 이루어진 연속된 데이터입니다. 
	예를 들어, 문장 "The cat sat on the mat"은 단어 단위로 나누어진 시퀀스로 볼 수 있습니다.
	입력 시퀀스는 시간적으로 순서가 중요한 데이터를 의미하며, 특히 NLP에서 사용될 때는 텍스트의 단어 또는 토큰들이 원래 문장 내에서 등장한 순서대로 배열된 것을 의미
	
	2. 출력 시퀀스: 트랜스포머 모델은 입력 시퀀스를 처리하여 출력 시퀀스를 생성합니다.
	예를 들어, 기계 번역 작업에서 입력 시퀀스가 영어 문장이라면, 출력 시퀀스는 해당 문장의 번역된 다른 언어 문장이 될 수 있습니다.
	
	3. 시퀀스 길이: 시퀀스는 고정된 길이가 아닐 수 있으며, 다양한 길이의 시퀀스를 처리할 수 있습니다.
	트랜스포머 모델은 이러한 가변 길이 시퀀스를 효율적으로 처리할 수 있도록 설계되었습니다.
	
	4. 포지셔널 인코딩(Positional Encoding): 트랜스포머는 순환 신경망(RNN)과 달리 시퀀스의 순차적 특성을 내재적으로 처리하지 않기 때문에, 
	입력 시퀀스의 각 위치에 대한 정보를 포지셔널 인코딩을 통해 추가합니다. 이는 모델이 시퀀스 내에서 각 토큰의 상대적 위치를 이해할 수 있도록 도와줍니다.
	
	5. 시퀀스 작업: 트랜스포머는 다양한 시퀀스 작업에 적용될 수 있습니다.
	예를 들어, 텍스트 분류, 시퀀스 라벨링, 기계 번역, 텍스트 생성 등 여러 NLP 작업에서 시퀀스를 입력 또는 출력으로 처리합니다.
	
	트랜스포머 모델은 이러한 시퀀스 데이터를 병렬로 처리할 수 있는 능력을 가지고 있으며, 이는 특히 긴 시퀀스를 다루는 작업에서 성능을 크게 향상시킵니다.
	이 모델은 어텐션 메커니즘을 통해 시퀀스 내의 모든 요소 간의 관계를 효과적으로 학습할 수 있도록 설계되었습니다.
	
	cp)
	입력 시퀀스는 입력된 시간 순서를 가진 텍스트를 의미합니다.
	
	NLP에서는 문장의 단어 또는 토큰들이 원래 순서를 유지한 상태로 전달됩니다.
	RNN/LSTM은 입력된 순서를 직접 활용하지만, Transformer 기반 모델은 위치 임베딩을 추가하여 시간 순서를 학습합니다.
	입력 시퀀스가 변경되면 문장의 의미가 달라질 수 있기 때문에, 원래 순서를 유지하는 것이 중요합니다.

- 시퀀스 투 시퀀스 모델(Sequence-to-Sequence, Seq2Seq)
	시퀀스 투 시퀀스 모델은 입력 시퀀스를 받아 출력 시퀀스를 생성하는 딥러닝 모델입니다.
	주로 자연어 처리(NLP) 분야에서 사용되며, 입력과 출력이 모두 시퀀스 형태인 작업에 적합합니다. 대표적인 예로는 기계 번역, 텍스트 요약, 질의 응답 시스템 등이 있습니다.
	
	Seq2Seq 모델의 기본 구조는 인코더(Encoder)와 디코더(Decoder)로 이루어져 있습니다.
	
	1. 인코더(Encoder): 입력 시퀀스를 받아 이를 고정된 길이의 컨텍스트 벡터(Context Vector)로 변환합니다. 
	인코더는 일반적으로 여러 층의 순환 신경망(RNN), LSTM(Long Short-Term Memory), 또는 GRU(Gated Recurrent Unit)로 구성됩니다. 
	입력 시퀀스의 각 요소를 처리하여 최종적으로 컨텍스트 벡터를 생성합니다.
	
	2. 디코더(Decoder): 인코더에서 생성된 컨텍스트 벡터를 받아 출력 시퀀스를 생성합니다. 
	디코더 역시 RNN, LSTM, 또는 GRU로 구성되며, 이전에 생성된 출력과 컨텍스트 벡터를 기반으로 다음 출력을 예측합니다.
	
	Seq2Seq 모델은 기본적으로 입력 시퀀스와 출력 시퀀스 간의 길이가 다를 수 있으며, 시퀀스의 길이가 정 해져 있지 않기 때문에 다양한 길이의 입력과 출력을 처리할 수 있습니다.
	또한, Seq2Seq 모델은 어텐션 메커니즘(Attention Mechanism)을 도입하여 성능을 향상시킬 수 있습니다. 
	어텐션은 인코더의 모든 출력 상태를 활용하여 디코더가 더 중요한 입력 부분에 집중할 수 있도록 도와줍니다.
	이를 통해 모델이 긴 시퀀스에서도 더 효과적으로 작동할 수 있게 됩니다.
	
	최근에는 트랜스포머(Transformer) 아키텍처가 Seq2Seq 모델의 성능을 크게 향상시켜, 많은 자연어 처리 작업에서 널리 사용되고 있습니다. 트랜스포머는 인코더-디코더 구조를 유지하면서 병렬 처리가 가능하고, 어텐션 메커니즘을 효과적으로 활용하여 빠르고 정확한 시퀀스 변환을 수행합니다.

- CNN(Convolutional Neural Network)
	CNN은 주로 이미지 인식 및 처리에 사용되는 딥러닝 모델로, 이미지의 공간적 및 시각적 패턴을 학습하는 데 특화되어 있습니다.
	CNN은 구조적으로 인간의 시각 피질을 모방하여 설계되었으며, 이미지의 특징을 자동으로 추출하고 학습할 수 있는 능력을 가지고 있습니다.
	
	▶︎ 기본 구조
	  CNN은 여러 종류의 층(layer)로 구성되며, 각 층은 데이터의 다른 수준의 추상화를 학습합니다. 주요 구성 요소는 다음과 같습니다
	
	1. 컨볼루션 층(Convolutional Layer)
	    • 이미지의 특징을 추출하는 역할을 합니다.
	    • 작은 크기의 필터(또는 커널)를 사용하여 입력 이미지의 특정 부분을 스캔하며, 이 과정을 통해 특징 맵(feature map)을 생성합니다.
	    • 필터는 가중치가 학습 가능한 매개변수로, 학습 과정에서 최적화됩니다.
	
	2. 활성화 함수(Activation Function)
	    • 비선형성을 추가하여 모델이 복잡한 패턴을 학습할 수 있도록 합니다.
	    • 가장 일반적으로 사용되는 활성화 함수는 ReLU(Rectified Linear Unit)입니다.
	
	3. 풀링 층(Pooling Layer)
	    • 특징 맵의 크기를 줄여 계산량을 감소시키고, 모델의 과적합을 방지합니다.
	    • 일반적으로 최대 풀링(max pooling)이나 평균 풀링(average pooling)이 사용됩니다.
	
	4. 완전 연결 층(Fully Connected Layer)
	    • 최종적으로 학습된 특징을 기반으로 분류를 수행합니다.
	    • 전통적인 신경망의 출력층과 유사하게 작동하며, 모든 입력 뉴런이 출력 뉴런과 연결됩니다.
	
	▶︎ 작동 방식
		• 입력 이미지가 각 층을 통과하면서 점점 더 추상적인 특징을 학습합니다.
		• 초기 층에서는 에지, 색상 등 기본적인 특징을 감지하고, 깊은 층으로 갈수록 더 복잡한 패턴이나 객체를 인식할 수 있습니다.
	
	▶︎ 특징
		• 자동 특징 추출: 수작업으로 특징을 추출할 필요 없이, 모델이 자동으로 이미지의 중요한 특징을 학습합니다.
		• 공간적 불변성: 이미지의 위치나 방향 변화에 대해 강건한 성능을 보입니다.
		•  효율성: 지역적 연결성을 이용하여 계산 효율성을 높입니다.
	
	▶︎ 응용 분야
		• 이미지 및 비디오 인식, 객체 탐지, 얼굴 인식, 자율 주행차의 시각 인식 시스템 등 다양한 분야에서 사용됩니다.
	
	CNN은 이미지 데이터의 복잡한 패턴을 효과적으로 학습할 수 있는 강력한 도구로, 딥러닝의 발전과 함께 많은 연구와 발전이 이루어지고 있습니다.
	특히, 대규모 데이터셋과 강력한 컴퓨팅 자원을 활용하여 높은 성능을 발휘할 수 있습니다.

- 시드
	모델을 학습할 때 재현성(reproducibility)을 보장하고, 결과의 일관성을 유지하기 위해 시드(seed)를 설정합니다. 시드는 난수(random number)를 결정하는 초기값입니다.
	LLM을 포함한 딥러닝 모델 학습에서는 가중치 초기화, 데이터 셔플링, 배치 선택 등에서 난수가 사용됩니다. 같은 시드를 설정하면 항상 동일한 난수를 생성하여 실험을 재현할 수 있습니다.
	
	▶︎ 시드의 필요성
	모델 훈련 과정에서 여러 요소가 **난수(randomness)**를 포함하고 있습니다:
	
	1) 가중치 초기화(Weight Initialization)
	신경망의 가중치는 일반적으로 랜덤 값으로 초기화됩니다.
	시드를 고정하면 같은 초기 가중치를 사용할 수 있습니다.
	
	2) 데이터 셔플링(Data Shuffling)
	배치(batch) 학습 시, 데이터가 랜덤으로 섞여야 일반화 성능이 향상됩니다.
	시드를 고정하면 항상 같은 순서로 셔플됩니다.
	
	3) 배치 샘플링(Mini-batch Sampling)
	미니배치를 랜덤으로 선택하는 과정에서도 난수가 사용됩니다.
	시드를 설정하면 같은 배치가 반복적으로 선택됩니다.
	
	4) 드롭아웃(Dropout)
	일부 뉴런을 무작위로 비활성화하는 드롭아웃도 난수를 사용합니다.
	시드를 설정하면 같은 뉴런이 드롭됩니다.
