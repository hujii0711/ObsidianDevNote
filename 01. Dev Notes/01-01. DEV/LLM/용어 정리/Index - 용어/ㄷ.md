- 대규모 언어 모델(Large Language Model)
	LLM은 고급 자연어 처리(NLP) 딥러닝 모델입니다. 이들은 대규모로 맥락적 언어를 처리하고 특정 언어에서 토큰 시퀀스의 가능성을 예측하는 데 특화되어 있습니다.
	의미의 가장 작은 단위인 토큰은 단어나 하위 단어일 수 있으며, LLM의 주요 입력으로 작용합니다.
	LLM은 자기회귀, 자동 인코딩, 또는 둘의 조합으로 분류될 수 있습니다. 이들의 가장 큰 특징은 방대한 크기로, 이를 통해 텍스트 생성과 분류와 같은 복잡한 언어 작업을 높은
	정밀도와 최소한의 파인튜닝으로 수행할 수 있습니다.

- 드롭아웃(Dropout)
	드롭아웃(Dropout)은 과적합(overfitting)을 방지하고 모델의 일반화 성능을 향상시키기 위해 사용되는 정규화 기법 중 하나입니다.
	드롭아웃은 학습 과정에서 임의로 일부 뉴런을 비활성화(즉, "드롭")하여 모델이 특정 뉴런이나 패턴에 지나치게 의존하지 않도록 합니다.
	
	드롭아웃의 주요 특징과 작동 방식은 다음과 같습니다:
	
	1. 임의의 뉴런 비활성화
	학습 과정에서 각 미니배치(mini-batch)마다 네트워크의 뉴런 중 일부를 확률적으로 비활성화합니다.
	일반적으로 드롭아웃 확률(p)을 설정하며, 이는 각 뉴런이 드롭아웃될 확률을 의미합니다. 예를 들어, p=0.5라면 각 뉴런이 50% 확률로 비활성화됩니다.
	
	2. 과적합 방지
	드롭아웃은 학습 과정에서 네트워크가 특정 뉴런에 의존하지 않도록 하여, 모델이 데이터의 노이즈나 특정 패턴에 과적합되는 것을 방지합니다.
	이는 모델이 더 일반화된 특성을 학습하도록 도와줍니다.
	
	3. 앙상블 효과
	드롭아웃은 학습 중에 다양한 하위 네트워크를 무작위로 생성하여 학습하는 효과를 가지므로, 일종의 앙상블 학습과 유사한 결과를 제공합니다.
	이는 모델의 예측 안정성을 높이고, 성능을 개선하는 데 기여합니다.
	
	4. 테스트 시 드롭아웃 비활성화
	학습이 끝난 후, 테스트나 추론 단계에서는 드롭아웃을 사용하지 않습니다.
	대신, 학습 시 비활성화된 뉴런의 효과를 보상하기 위해 각 뉴런의 출력을 드롭아웃 확률(p)로 곱하여 조정합니다.
	이는 학습 중 드롭아웃된 뉴런의 평균적 효과를 고려하여, 테스트 시 일관된 출력을 보장합니다.
	
	드롭아웃은 특히 대규모 네트워크에서 과적합을 줄이는 데 효과적이며, 다양한 딥러닝 모델에서 널리 사용되고 있는 정규화 기법입니다. 이를 통해 모델의 성능과 일반화 능력을 향상시킬 수 있습니다.
