- 배치 크기(Batch size)
	배치 크기는 모델이 한 번에 학습하는 훈련 예제의 수를 나타냅니다.
	더 큰 배치 크기는 더 빠르지만 덜 상세한 학습을 의미할 수 있고, 반면 더 작은 배치 크기는 더 느리지만 잠재적으로 더 상세한 이해로 이어질 수 있습니다.
	실제 인공지는 모델 훈련 시에는 단일 글자 텐서만 입력으로 주어지지 않고 여러 개의 텐서가 함께 묶여 입력으로 제공된다. 이를 배치하고 한다.
	배치 처리는 모델의 학습 효율을 높이는 중요한 기법으로, 여러 데이터 샘플을 동시에 처리함으로서 게산 속도를 향상시키고 모델의 일반화 능력을 개선한다.

- BM25
	BM25는 문서 내 쿼리 용어의 빈도에 따라 결과를 순위화하고 용어의 근접성과 역문서 빈도를 고려합니다.
	BM25는 전체 쿼리의 문맥을 고려하지 않지만, 검색 결과를 재순위화하고 결과의 전반적인 관련성을 향상시키는데 유용한 방법일 수 있습니다.

- 벡터
	텍스트 데이터를 컴퓨터가 이해하고 처리할 수 있는 숫자 형태
	임베딩을 통해 단어를 숫자 벡터로 변환하면 컴퓨터가 이해할 수 있는 형태가됨
	벡터 표현은 다양한 수학적 연산을 가능하게 해 복잡한 언어 모델링과 자연처 처리 작업에 유용
