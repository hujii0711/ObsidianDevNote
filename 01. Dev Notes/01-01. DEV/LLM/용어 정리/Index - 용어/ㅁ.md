- 말뭉치(Corpora, Corpus)
	말뭉치 또는 코퍼스는 텍스트 데이터의 모음입니다. 연구자들의 자원, 자료에 비유할 수 있습니다. 말뭉치의 질과 양이 좋을수록 LLM은 더 잘 학습할 수 있습니다.

- 멀티모달(Multimodal)
	멀티모달(Multimodal) 학습은 여러 가지 형태(modality)의 데이터를 동시에 활용하여 모델을 학습시키고, 더 풍부하고 정확한 정보를 추출하는 방법을 의미합니다.
	서로 다른 형태의 데이터는 텍스트, 이미지, 오디오, 비디오 등 다양한 유형을 포함할 수 있으며, 이러한 데이터들을 결합하여 더 나은 성능을 달성하는 것을 목표로 합니다.
	
	멀티모달 학습의 주요 개념과 응용 분야는 다음과 같습니다:
	
	1. 데이터의 다양성 활용
	멀티모달 접근 방식은 각 데이터 형태가 제공하는 고유한 정보를 활용합니다. 예를 들어, 이미지 데이터는 시각적 정보를 제공하고, 텍스트 데이터는 언어적 맥락을 제공합니다. 이러한 다양한 정보의 결합은 모델이 더 깊은 이해를 갖도록 돕습니다.
	
	2. 상호 보완적 정보
	서로 다른 데이터 형태는 종종 상호 보완적입니다. 예를 들어, 비디오 데이터에서 오디오는 장면의 분위기나 대화를 이해하는 데 도움을 줄 수 있으며, 텍스트는 이미지에 대한 설명을 제공할 수 있습니다.
	
	3. 응용 분야
	   • 이미지 캡셔닝: 이미지에 대한 설명을 자동으로 생성하는 작업으로, 이미지와 텍스트 데이터를 결합하여 수행됩니다.
	   • 비디오 분석: 비디오에서 시각적 정보와 오디오 정보를 결합하여 더 나은 행동 인식이나 이벤트 감지를 수행합니다.
	   • 감정 인식: 얼굴 표정, 음성 톤, 텍스트 등을 결합하여 사람의 감정을 인식하는 데 사용됩니다.
	   • 자율 주행: 자율 주행 차량은 카메라, 라이다, GPS 등 여러 센서 데이터를 결합하여 주행 환경을 인식하고, 안전하게 주행할 수 있습니다.
	
	4. 모델 설계
	멀티모달 학습을 위해서는 각 모달리티의 데이터를 효과적으로 처리하고 결합할 수 있는 모델 설계가 필요합니다. 이는 각 모달리티의 특징을 추출하고, 이들을 적절히 융합하는 방법을 포함합니다.
	
	멀티모달 학습은 다양한 형태의 데이터를 통합하여, 단일 모달리티로는 얻을 수 없는 풍부한 정보를 제공함으로써, 딥러닝 모델의 성능을 향상시키고, 더 복잡한 문제를 해결할 수 있도록 합니다.

- 마스크드 언어 모델(MLM)이란?
	입력 문장에서 일부 단어를 [MASK](모델이 문장에서 일부 단어를 가리고(masking)) 토큰으로 대체한 후, 모델이 해당 단어를 예측하도록 학습하는 방식
	대표적인 예시: BERT(Bidirectional Encoder Representations from Transformers)
	
	1. 마스크된 언어 모델의 특징
	• 양방향 학습
	기존의 RNN 기반 언어 모델은 좌→우 또는 우→좌 방향으로만 학습
	하지만 MLM은 문장의 양쪽(앞뒤)을 모두 고려하여 마스크된 단어를 예측
	
	• 완전한 문맥 이해
	다음 단어를 예측하는 GPT 방식(오토리그레시브)과 달리, 문장 전체의 의미를 파악하는 데 유리
	
	• 사전 훈련(pretraining) 방식
	먼저 MLM 방식으로 모델을 훈련한 후, 특정 태스크(QnA, 문서 분류 등)에 맞게 미세 조정(fine-tuning)
	
	2. 학습 과정
	• 입력
	오늘은 [MASK] 날씨가 좋다.
	
	• 모델 예측
	오늘은 정말 날씨가 좋다.
	오늘은 매우 날씨가 좋다.
	
	BERT 같은 모델은 [MASK] 자리에 가장 적절한 단어를 예측하는 방식으로 학습됩니다.
