- 컴퓨터 비전(Computer Vision)
	컴퓨터 비전(Computer Vision)은 컴퓨터가 디지털 이미지나 비디오에서 의미 있는 정보를 추출하고 해석할 수 있도록 하는 기술 및 연구 분야를 의미합니다.
	컴퓨터 비전은 인간의 시각적 인지 능력을 모방하여, 컴퓨터가 시각적인 데이터를 이해하고 처리할 수 있도록 하는 것을 목표로 합니다.
	딥러닝은 컴퓨터 비전의 발전에 큰 기여를 하고 있으며, 여러 복잡한 문제를 해결하는 데 사용됩니다.
	
	컴퓨터 비전의 주요 목표와 응용 분야는 다음과 같습니다:
	
	1. 이미지 분류
	 주어진 이미지가 어떤 클래스에 속하는지를 예측하는 작업입니다. 예를 들어, 이미지가 고양이인지 개인지 분류하는 문제를 해결합니다.
	
	2. 객체 탐지
	이미지나 비디오에서 특정 객체의 위치를 식별하고, 해당 객체가 무엇인지를 예측하는 작업입니다. 이는 자율 주행 자동차에서 보행자나 차량을 탐지하는 데 사용됩니다.
	
	3. 이미지 세분화
	이미지의 각 픽셀을 특정 클래스에 할당하는 작업입니다. 이는 의료 영상에서 종양을 식별하거나, 위성 이미지에서 특정 지형을 구분하는 데 활용됩니다.
	
	4. 얼굴 인식
	이미지나 비디오에서 사람의 얼굴을 인식하고 식별하는 기술입니다. 이는 보안 시스템이나 소셜 미디어에서 사용자 태그에 사용됩니다.
	
	5. 행동 인식
	비디오에서 사람의 행동이나 활동을 인식하는 작업입니다. 이는 감시 시스템이나 스포츠 분석에 활용됩니다.
	
	6. 이미지 생성 및 변환
	GAN(Generative Adversarial Networks)과 같은 딥러닝 모델을 사용하여 새로운 이미지를 생성하거나, 이미지 스타일을 변환하는 작업입니다.
	
	컴퓨터 비전 분야에서 딥러닝은 특히 CNN(Convolutional Neural Networks)과 같은 구조를 사용하여  이미지의 특징을 자동으로 추출하고 학습하는 데 큰 성과를 거두었습니다.
	이로 인해, 컴퓨터 비전 기술은 다양한 산업과 응용 분야에서 혁신적인 변화를 이끌고 있습니다.

- KNN(K-Nearest Neighbors)
	KNN은 가장 단순하면서도 널리 사용되는 지도 학습 알고리즘 중 하나입니다.
	주로 분류와 회귀 문제를 해결하는 데 사용되며, 데이터의 패턴을 학습하기보다는 데이터를 저장하고 그 주변 이웃을 통해 의사 결정을 합니다.
	
	▶︎ 기본 개념
	  • KNN은 새로운 데이터 포인트의 클래스나 값을 결정하기 위해 가장 가까운 K개의 이웃을 참조합니다. 이웃의 다수결에 따라 새로운 데이터 포인트의 클래스를 결정하거나, 이웃의 평균값을 통해 값을 예측합니다.
	
	▶︎ 작동 방식
	  1. 거리 계산: 새로운 데이터 포인트와 모든 기존 데이터 포인트 간의 거리를 계산합니다. 일반적으로 유클리드 거리(Euclidean distance)를 사용하지만, 맨해튼 거리(Manhattan distance)나 코사인 유사도(Cosine similarity) 등도 사용할 수 있습니다.
	  2. 이웃 선택: 계산된 거리 중 가장 가까운 K개의 이웃을 선택합니다.
	  3. 클래스 결정(분류의 경우): 선택된 K개의 이웃 중 가장 많이 등장하는 클래스를 새로운 데이터 포인트의 클래스로 지정합니다.
	  4. 값 예측(회귀의 경우): 선택된 K개의 이웃의 값을 평균하여 새로운 데이터 포인트의 값을 예측합니다.
	
	▶︎ 하이퍼파라미터
	  • K의 값: K의 크기는 모델의 성능에 큰 영향을 미칩니다. 너무 작으면(예: K=1), 모델이 데이터의 노이즈에 민감해질 수 있고, 너무 크면 다양한 클래스의 데이터가 섞여 정확도가 떨어질 수 있습니다. 일반적으로 홀수 값을 사용하여 동점 상황을 피합니다.
	
	▶︎ 특징
	  • 비모수적: KNN은 명시적인 학습 단계가 없으며, 예측 시점에 계산이 이루어집니다.
	  • 단순성: 이해하고 구현하기 쉬운 알고리즘입니다.
	  • 유연성: 다양한 거리 측정 방법을 사용할 수 있어 다양한 유형의 데이터에 적용 가능합니다.
	
	▶︎ 단점
	  • 계산 비용: 데이터셋이 클 경우, 거리 계산에 많은 시간이 소요될 수 있습니다.
	  • 메모리 사용량: 모든 데이터를 저장해야 하므로 메모리 사용량이 많습니다.
	  • 고차원 데이터: 차원의 저주(curse of dimensionality) 문제로 인해 고차원 데이터에서는 성능이 저하될 수 있습니다.
	
	KNN은 그 단순성과 직관성 덕분에 다양한 분야에서 사용되며, 특히 데이터의 구조나 분포에 대한 사전 지식 이 없을 때 유용합니다. 또한, 데이터 전처리와 특징 선택이 모델의 성능에 큰 영향을 미칠 수 있기 때문에, 이를 신중하게 고려해야 합니다.
