
#### 1. 모델(설정, 로드, 트랜스포머)
- 임베딩
- 인코더
- 디코더
- 어텐션, 셀프 어텐션, 멀티 헤드 어텐션, 마스크드 멀티 헤드 어텐션
	입력 시퀀스 내의 모든 단어 간의 관계를 학습
- 순방향 신경망
	각 단어의 특징을 추출해 다음 계층으로 전달한다.
- RNN(순환 신경망, Recurrent Neural Networks)
- LSTM(장단기 기억, Long Short-Term Memory)
- 위치 인코딩
- 히든 유닛수
- 어텐션 헤드 수
- 드롭아웃
- 순전파
	전처리된 입력 데이터를 모델에 전달해 출력을 생성한다. 이 과정에서 모델의 계산 그래프가 실행된다.
- 마스크된 언어 모델(MLM)
- 다음 문장 예측(NSP)
- 잔차 연결(Residual connection)
- 기울기 소실(Vanishing gradient)
- ReLU(Rectified linear unit)
	활성화 함수
- SwiGLU
	Llama 3에서 사용하는 활성화 함수

#### 2. 토크나이저
- 서브워드
- 토큰화
- 잘라내기
- 패딩
- 마스킹
- 텍스트 정규화
- 대소문자 변환
- 구두점 제거
- 불용어 처리
- 입력 ID(input_ids)
	입력 텍스트를 정수 인코딩으로 변환한 값, 이는 텍스트의 각 토큰을 고유한 정수로 변환해 모델에 입력으로 사용될 수 있도록 한다.
- 세그먼트 ID(token_type_ids)
	입력이 여러 세크먼트로 구성된 경우 각 세그먼트를 구분하는 값을 제공한다.
- 어텐션 마스크(attension_mask)
	트랜스포머 인코더의 셀프 어텐션에 사용되는 마스크 값을 의미하며, 이는 모델이 어떤 토큰을 무시해야 하는지를 지정하는 역할
- BPE 토크나이저
