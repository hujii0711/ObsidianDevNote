- 전이학습
	사전 학습된 모델의 가중치를 그대로 사용하거나 일부만 조정하여 새로운 작업에 적용하는 방법
	
- 선형(Linear)
	선형함수는 입력과 출력 사이의 관계가 직선 형태로 유지되는 함수
	
- 비선형(Non-Linear)
	비선형함수는 입력과 출력 사이의 관계가 단순한 직선으로 표현되지 않는 함수
	선경망에서 활성화 함수가 비선형을 추가하는 역할
	비선형함수를 사용해야 딥러닝 모델이 복잡한 패턴을 학습할 수 있다
	
- 시퀀스 데이터
	순서가 중요한 데이터가 나열된 순서에 따라 의미가 달라지며, 시간적 공간적 의존성이 있는 데이터이다.
	
- 시계열 데이터
	시간의 흐름에 따라 수집된 데이터(시간 순서가 중요한 특징)
	
- 손실함수
	예측값과 실제값의 차이를 계산. 손실을 기반으로 가중치에 대한 기울기(gradient)를 계산한다.
	
- 역전파(Background Propagation)
	모델의 예측값과 실제값의 차이를 계산하고, 가중치를 업데이트하는 과정
- 역전파의 목적
	신경망이 예측한 값과 실제값 사이의 차이를 줄이도록 가중치를 조정한다.
	
- 과적합 = 과대적합(Overfitting)
	모델이 훈련 데이터를 과하게 학습하여 보지 못한 데이터나 테스트 데이터에서 성능이 떨어진 상태를 말한다.
	
- 과소적합(Underfitting)
	모델이 훈련 데이터에 있는 기본 패턴을 포착하기에 너무 단순하여 훈련 및 테스트 데이터 모두에서 성능이 떨어진 상태이다.
	
- 개체명 인식(NER)
	NLP의 한 과제로 텍스트에서 특정한 개체명(사람, 장소, 조직, 날짜 등)을 식별하고 분류하는 작업이다.
	
- 교차 엔트로피 손실 함수(Cross-Entropy Loss)
	분류 문제이서 모델의 예측과 실제 레이블 간의 차이를 측정하는데 사용하는 손실함수이다.
	
- 그레이디언트(Gradient)
	모델이 정답에 가까워지려면 어느 방향으로 얼마나 움직여야 하는지 알려주는 값이다.
	
- 드롭아웃(Dropout)
	과적합을 방지하고 모델의 일반화 성능을 향상시키기 위해 사용되는 정규화 기법이다.
	
- 로짓(Logits)
	모델의 학습 결과의 최종 출력으로서 확률로 변환되기 전의 원시 점수의 값이다.
	
- 소프트맥스(Softmax)
	Logits은 소프트맥스와 같은 활성화 함수를 통해 확률 분포로 변환한다. 모델의 출력을 확률 분포로 변환한다.
	
- 시퀀스
	딥러닝 모델인 트랜스포머에서 주로 연속된 데이터 포인트의 집합을 의미한다.
	단어, 토큰 또는 문장으로 이루어진 연속된 데이터이다. -> 입력 시퀀스
	
- 출력 시퀀스
	입력 시퀀스를 처리하여 출력 시퀀스를 생성한다.(영어문장(입력시퀀스) -> 한글문장(출력시퀀스), 이를 시퀀스 투 시퀀스라고 함)
	
- 인코더
	입력 시퀀스를 받아 이를 고정된 길이의 컨텍스트 벡터로 변환한다.
	(RNN, LSTM, GRU로 구성)
	
- 디코더
	인코더에서 생성된 컨텍스트 벡터를 받아 출력 시퀀스를 생성한다.
	(RNN, LSTM, GRU로 구성)
	
- 시드
	모델을 학습할 때 재현성을 보장하고, 결과의 일관성을 유지하기 위해 시드를 생성한다.
	시드는 난수를 결정하는 초기값이다.
	
- 원 핫 인코딩
	모델이 범주형 데이터(클래스 -> ex. red, green, blue)를 처리할 수 있도록 범주형 데이터를 수치형 데이터로 변환하는 기법 중 하나이다.
	
- 엔드 투 엔드 학습
	데이터 입력부터 최종 출력까지의 모든 단계를 하나의 모델로 학습하는 방법을 의미한다.
	
- 자기 회귀 언어 모델
	시퀀스에서 이전 토큰에만 기반하여 문장의 다음 토큰을 예측한다.
	
- 청크
	데이터를 더 작은 조각으로 나누는 것을 의미한다.
	
- 토큰
	토큰은 의미를 가지는 가장 작은 단위로, 문장이나 텍스트를 작은 단위로 나누어 생성하는 것이다.
	
- 텐서
	텐서는 딥러닝에서 데이터를 표현하는 기본 단위로 다차원(숫자) 배열 형태로 데이터를 저장하고 처리한다.
	
- 패딩
	텍스트 길이를 통일하여 모든 데이터를 모델의 최대 길이나 정해진 길이로 맞추는 작업이다.
	데이터 입력 길이를 고정할 수 있어 모델의 병렬화가 용이해진다.
	
- 하이퍼 파라미터
	모델 훈련 과정에서 조정할 수 있는 설정이다.
	
- ANN (Artifical Neural Network)
	
- ANN(Approximate Nearest Neighbor)
	데이터베이스에서 주어진 쿼리 포인트에 대해 가장 가까운 데이터 포인트를 빠르게 찾기 위한 방법론을 의미한다.
	근사O, 최근접 이웃을 빠르게 찾기
	
- KNN(K-Nearest Neighbors)
	새로운 데이터 포인트의 클래스나 값을 결정하기 위해 가장 가까운 K개의 이웃을 참조한다. 이웃의 다수결에 따라 새로운 데이터 포인트의 클래스를 결정하거나 이웃의 평균값을 통해 값을 예측한다.
	-> 유클리드 거리, 코사인 유사도등 사용하여 계산된 거리 중 가장 가까운 K개의 이웃을 선택한다.
	
- RNN(순환 신경망, Recurrent Neural Network)
	시계열이나 데이터나 순차적 데이터에 적합한 신경망 구조로, 데이터의 순서나 시각적 패턴을 학습하는데 특화되었다.
	
- CNN(Convolution Neural Network)
	이미지 인식 및 처리에 사용되는 딥러닝 모델로, 이미지의 공간적 및 시각적 패턴을 학습하는데 특화되었다.
	
- AGI(Artifical General Inteligence)
	일반 인공 지능, 범용 인공 지능
	-> 인간이 수행할 수 있는 모든 지적인 작업을 이해하고 학습
	
- AI Agent
	단순히 질문에 답변하는 챗봇, LLM보다 한 단계 더 나아가 추론, 기억, 계획을 기반으로 실제 행동(Action)을 실행할 수 있는 능력을 갖춘 시스템이다.
