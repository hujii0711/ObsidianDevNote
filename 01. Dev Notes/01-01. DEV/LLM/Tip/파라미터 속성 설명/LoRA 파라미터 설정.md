```python
peft_config = LoraConfig(
	lora_alpha=128,
	lora_dropout=0.05, 
	r=256,
	bias="none",
	target_modules=[
		"q_proj",
		"o_proj",
		"k_proj",
		"v_proj",
		"up_proj",
		"down_proj",
		"gate_proj",
	],
	task_type="CAUSAL_LM"
)
```

1. lora_alpha
	학습 속도 설정

2. lora_dropout
	Lora 학습 때 사용할 dropout 확률을 지정합니다. 드롭아웃 확률은 과적합 방지를 위해 학습 중 무작위로 일부 뉴런을 비활성화하는 비율을 지정합니다.

3. r
	Lora의 저차원 공간의 랭크를 지정합니다. 랭크가 높을수록 모델의 표현력이 증가하지만, 계산 비용도 증가합니다. 모델의 표현력을 결정하는 랭크를 지정

4.  bias
	Lora의 저차원 공간의 랭크를 지정합니다. 랭크가 높을수록 모델의 표현력이 증가하지만, 계산 비용도 증가합니다.

5. target_modules
	LoRA 설정에서 모델의 핵심 부분들만 선택적으로 선택하여 target_modules로 지정
	이런 방식은 메모리 효율을 높이고, 학습 속도를 향상시키며, 과적합 위험을 줄이고, 모델의 핵심 기능을 효과적으로 조정하면서도
	전체적인 학습 효율설을 크게 높일 수 있다.
		4-1. q_proj: 쿼리는 특정 단어에 대한 정보 요청
		4-2. k_proj: 키는 다른 단어들의 정보를 나타냄. 쿼리와 키 둘의 상호 작용을 통해 단어 간 관련성을 결정한다.
		4-3. v_proj: 각 단어의 실제 정보를 나타낸다.
		4-4. o_proj: 셀프 어텐션 메커니즘의 결과를 종합해 최종 출력을 만든다. Lora를 적용할 모델의 모듈 리스트입니다.
		4-5. up_proj, down_proj: 정보를 더 깊은 층으로 전달하거나 더 표면적인 층으로 전달하는 역할
		4-6. gate_proj: 정보의 중요도를 결정하는 게이트 기능을 수행

6. task_type
	모델의 학습 방식으로 지정하는 중요한 매개변수이다.
	CAUSAL_LM은 인과적 모델 언어를 의미한다. 인과적이라는 표현은 모델이 오직 이전에 등장한 단어들만 고려해 예측한다는 특성을 나타낸다.
	이 설정을 사용하면 지금까지 본 정보만을 바탕으로 다음에 올 내용을 예측하라고 모델에 지시하는 것과 같다. 미세 조정 작업 유형을 CAUSAL_LM으로 지정하여 언어 모델링 작업을 수행합니다.
