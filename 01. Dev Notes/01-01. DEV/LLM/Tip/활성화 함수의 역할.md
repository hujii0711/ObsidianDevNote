
LLM(Large Language Model)을 파인튜닝(fine-tuning)할 때 활성화 함수(activation function)는 신경망의 비선형성(non-linearity)을 도입하여 모델이 더 복잡하고 풍부한 패턴을 학습할 수 있도록 하는 중요한 구성 요소입니다. 아래에 활성화 함수의 구체적인 역할과 LLM 파인튜닝과의 관계를 설명드립니다:

### 1. 비선형성(introducing non-linearity)
- 인공신경망은 여러 계층을 거쳐 데이터를 처리합니다.
- 활성화 함수를 사용하지 않으면 각 층이 선형 변환(linear transformation)만 수행하게 되어, 아무리 깊은 층이 있어도 전체 네트워크도 결국 하나의 선형 함수가 됩니다.
- 활성화 함수는 이러한 한계를 극복하고, 언어와 같은 복잡한 입력에 존재하는 비선형 패턴을 학습할 수 있게 합니다.

### 2. 표현력 강화 (Improved Representational Power)
- LLM은 자연어의 복잡한 의미, 구문, 맥락 등을 모델링해야 합니다.
- 이를 위해선 모델이 데이터의 다양한 비선형 관계를 포착할 수 있어야 하며, 활성화 함수가 이 기능을 담당합니다.
- 예: 단어 간의 비직접적인 의미 관계나 문맥에 따른 단어 의미 변화 등

### 3. 정보 흐름 조절
- 활성화 함수는 특정 뉴런의 출력이 다음 층에 전달될지를 결정합니다. 예를 들어 ReLU는 입력이 0 이하인 경우 출력을 0으로 만들기 때문에, 네트워크의 sparsity(희소성)를 증가시켜 학습 효율을 높일 수 있습니다.
- 이는 파인튜닝 과정에서도 유효하며, LLM의 일부 계층만 미세 조정할 때도 중요한 역할을 합니다.

### 4. 활성화 함수의 종류와 특성
LLM 및 트랜스포머 아키텍처에서는 주로 다음과 같은 활성화 함수가 사용됩니다:

- ReLU (Rectified Linear Unit)
  - 가장 기본적인 활성화 함수
  - 음수는 0으로, 양수는 그대로 반환
  - 빠르고 단순함
- GELU (Gaussian Error Linear Unit)
  - BERT, GPT 등에서 사용
  - 입력값을 확률적으로 조정, 더 부드러운 비선형성 제공
  - 성능 향상 측면에서 ReLU보다 우수한 경우가 많음

  GELU(x) ≈ 0.5 x (1 + tanh(√(2/π)·(x + 0.044715·x³)))

- Swish, SiLU 등
  - 최근 더 부드러운 버전의 활성화 함수도 실험적으로 사용
  - Swish(x) = x · sigmoid(x)

### 5. 파인튜닝에서의 적용 예
파인튜닝은 기존 LLM의 가중치를 기반으로 특정 태스크(예: 감정 분석, 질의응답 등)에 맞게 추가 학습하는 과정입니다. 

- 기존 모델의 활성화 함수는 유지되며, 파인튜닝 대상 레이어의 출력/입력 흐름에 영향을 줍니다.
- 예컨대 GPT나 BERT를 파인튜닝할 때, feedforward layer의 활성화 함수(GELU 등)는 그대로 사용되며 새롭게 추가된 머리부분(classification head)에서는 ReLU나 Tanh 등의 함수를 선택적으로 사용할 수 있습니다.

요약)
파인튜닝 시, 활성화 함수의 역할은 다음과 같습니다:
1. 모델에 비선형성 제공
2. 표현력 향상
3. 특정 뉴런의 활성화 여부 결정 → 학습 효율성 개선
4. 모델 출력의 의미를 더 잘 반영하도록 도와줌
5. 미세조정(fine-tuning) 시 성능에 간접적으로 큰 영향을 미침

따라서 활성화 함수는 파인튜닝에서 직접 수정하는 경우가 흔하진 않지만, 모델이 특정 작업에 얼마나 적합하게 적응하는지를 결정하는 데에 중요한 역할을 합니다.
