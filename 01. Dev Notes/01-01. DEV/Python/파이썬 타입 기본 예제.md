## 파이썬 타입 힌트 기본 예제

### 1. 기본 타입
```python
# 변수
name: str = "Alice"
age: int = 30
height: float = 1.75
is_active: bool = True

# 함수
def greet(name: str, age: int) -> str:
    return f"안녕하세요, {name}님! 나이: {age}"
```

---

### 2. 컬렉션 타입
```python
# Python 3.9+ 는 소문자로 직접 사용 가능
names: list[str] = ["Alice", "Bob"]
scores: dict[str, int] = {"Alice": 95, "Bob": 87}
unique_ids: set[int] = {1, 2, 3}
point: tuple[int, int] = (10, 20)

# 고정 길이 튜플
rgb: tuple[int, int, int] = (255, 128, 0)
```

---

### 3. Optional (None 허용)
```python
from typing import Optional

# 아래 두 표현은 동일 (Python 3.10+는 | 권장)
def find_user(user_id: int) -> Optional[str]:
    ...

def find_user(user_id: int) -> str | None:  # 권장
    if user_id == 1:
        return "Alice"
    return None
```

---

### 4. Union (여러 타입 허용)
```python
# Python 3.10+
def process(value: int | str | float) -> str:
    return str(value)

# Python 3.9 이하
from typing import Union
def process(value: Union[int, str, float]) -> str:
    return str(value)
```

### 6. Callable (함수 타입)
```python
from typing import Callable

# 인자 없고 int 반환
def run(func: Callable[[], int]) -> int:
    return func()

# int 두 개 받고 bool 반환
def filter_data(
    data: list[int],
    predicate: Callable[[int], bool]
) -> list[int]:
    return [x for x in data if predicate(x)]

# 사용 예
result = filter_data([1, 2, 3, 4], lambda x: x > 2)
```

### 7. TypedDict (딕셔너리 구조 정의)
```python
from typing import TypedDict

class User(TypedDict):
    name: str
    age: int
    email: str

user: User = {
    "name": "Alice",
    "age": 30,
    "email": "alice@example.com"
}
```

### 8. 클래스 타입
```python
class Animal:
    name: str
    age: int

    def __init__(self, name: str, age: int) -> None:
        self.name = name
        self.age = age

    def speak(self) -> str:
        return f"{self.name} says hello"

def process_animal(animal: Animal) -> str:
    return animal.speak()
```

### 9. TypeVar (제네릭)
```python
from typing import TypeVar

T = TypeVar("T")

def first(items: list[T]) -> T:
    return items[0]

first([1, 2, 3])        # int 반환
first(["a", "b", "c"])  # str 반환
```

### 10. Final / Literal
```python
from typing import Final, Literal

# 상수 (재할당 불가)
MAX_SIZE: Final = 100

# 특정 값만 허용
Direction = Literal["left", "right", "up", "down"]

def move(direction: Direction) -> None:
    print(f"Moving {direction}")

move("left")   # OK
move("diagonal")  # mypy 오류
```

### 버전별 요약

| 문법 | Python 3.9+ | Python 3.10+ |
|------|-------------|--------------|
| 리스트 | `list[str]` | `list[str]` |
| 옵셔널 | `str \| None` | `str \| None` ✅ 권장 |
| 유니온 | `Union[int, str]` | `int \| str` ✅ 권장 |

3.10+ 환경이라면 `from typing import Union, Optional` 없이 `|` 문법만으로 대부분 해결됩니다.
