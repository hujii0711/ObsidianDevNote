
### 1. **Mypy** — 가장 널리 쓰이는 표준
```bash
pip install mypy
mypy your_file.py
```
- Python 공식 생태계와 가장 밀접
- 설정 유연함 (`mypy.ini`, `pyproject.toml`)
- 엄격 모드: `mypy --strict your_file.py`
- **추천 대상:** 대부분의 프로젝트, 처음 도입 시

---

### 2. **Pyright / Pylance** — 빠르고 강력함
```bash
pip install pyright
pyright your_file.py
```
- Microsoft 개발, VS Code의 Pylance 확장이 이걸 기반으로 함
- Mypy보다 빠른 분석 속도
- 추론 능력이 더 정교한 경우 많음
- **추천 대상:** VS Code 사용자, 대형 코드베이스

---

### 3. **Ruff** — 린터 + 일부 타입 체크
```bash
pip install ruff
ruff check your_file.py
```
- 주목적은 린터이지만 타입 관련 규칙도 포함
- 매우 빠름 (Rust 기반)
- **추천 대상:** 타입 체크보다 코드 품질 전반 관리 시

---

### 4. **Pydantic** — 런타임 타입 검증
```python
from pydantic import BaseModel

class User(BaseModel):
    name: str
    age: int

user = User(name="Alice", age="30")  # 자동 변환 또는 오류
```

- 정적 분석이 아닌 **실행 시점** 검증
- API, 설정 파일, 데이터 파싱에 특히 유용
- **추천 대상:** FastAPI, 데이터 유효성 검사

## 조합 추천

| 상황 | 추천 조합 |
|------|-----------|
| 일반 프로젝트 | Mypy + Ruff |
| VS Code 환경 | Pylance + Mypy |
| FastAPI/웹 개발 | Pyright + Pydantic |
| CI/CD 파이프라인 | Mypy `--strict` |

처음 도입한다면 **Mypy**부터 시작하고, VS Code를 쓴다면 **Pylance**를 함께 켜두는 게 가장 무난합니다.

## Ruff 도입 예제

### 프로젝트 구조

my_project/
├── pyproject.toml
├── src/
│   └── main.py

    └── test_main.py

### 1. 설치
```bash
pip install ruff
```

---

### 2. `pyproject.toml` 설정
```toml
[tool.ruff]
line-length = 88
target-version = "py311"

[tool.ruff.lint]
select = [
    "E",   # pycodestyle 오류
    "W",   # pycodestyle 경고
    "F",   # pyflakes (미사용 변수, import 등)
    "I",   # isort (import 정렬)
    "B",   # flake8-bugbear (잠재적 버그)
    "UP",  # pyupgrade (최신 문법 권장)
]
ignore = [
    "E501",  # 줄 길이 (line-length로 대체)
]

[tool.ruff.lint.isort]
known-first-party = ["my_project"]
```

### 4. Ruff 실행
```bash
# 검사
ruff check src/

# 자동 수정
ruff check --fix src/

# 포맷팅 (Black 대체)
ruff format src/
```

### 6. CI/CD 연동 (GitHub Actions)
```yaml
# .github/workflows/lint.yml
name: Lint

on: [push, pull_request]

jobs:
  ruff:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: "3.11"
      - run: pip install ruff
      - run: ruff check .
      - run: ruff format --check .
```
